# Intruções de compilação e execução

Primeiro para que você possa testar o nosso programa, aconselhamos baixar o VScode(Visual Studio Code) e também a versão mais recente do .NET, até então o programa está rodando somente no Terminal do codigo. <br>

Após baixar inicie o VScode e entre na pasta do programa e pelo terminal selecione o diretorio correto do (./Codigo/). Agora basta rodar o codigo digitando "dotnet run" (Certifique-se de que você tem o .NET baixado no seu PC.
